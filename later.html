Google Later — A Chrome Extension for Productivity

Firebase Demo

Later is a chrome extension I started in 2017 and it is meant to be a desktop version of Google Now

I really enjoyed the functionality of Now cards on android and found that the extension I used for my new tab, while pretty, lacked functionality. So I attempted to make a new tab that was nice to look at, customisable, and useful.

version 0.5

So far, I'm still building the base app. The first few iterations just added in cards, set up the layout of the app, and set up accounts and other stuff necessary for all the apis used.

So far these are the cards I have made that work

  • News Gets the top stories from Google News. I plan on adding tabs to cycle to different news sources as the API allows it and it would be more useful
  • Weather Gets the weather of the current location. I'm adding functionality to have multiple weather cards for manually chosen locations
  • Reddit Top posts from Reddit. Just figured it would be cool to have
  • Search For a quick google search even though most people will just use the chrome url bar
  • Bookmarks/Links Allows you to import you bookmarks from chrome and/or set quick links
